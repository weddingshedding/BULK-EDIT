# BulkPix Studio

A fully client-side bulk photo editor for GitHub Pages.

## Main features

- Select individual photos or a complete folder
- Up to 1000 photos in one batch
- Default output: 400 × 400 px
- Default maximum size: 100 KB
- JPG, WebP and PNG output
- Fit, crop/fill and stretch modes
- Manual editing for each photo
- Rotate, flip, zoom, position, brightness, contrast, saturation, grayscale, blur and sharpness
- All processed photos download together in one ZIP
- No login, database, Firebase, cloud upload or permanent storage

## GitHub Pages deployment

1. Create a public GitHub repository.
2. Upload `index.html`, `README.md` and `.nojekyll` to the repository root.
3. Open **Settings → Pages**.
4. Select **Deploy from a branch**.
5. Select **main** and **/(root)**, then save.

The app works entirely inside the browser. For very large batches, desktop Chrome is recommended because phones have less memory.
